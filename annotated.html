The output, y_o, represents the Manchester-encoded signal. @structure The entity has the following ports: `clk_i` is the clock input, used to synchronize the circuit. `rst_i` is the reset input, which resets the state machine and other internal signals when asserted. `v_i` is a signal that indicates whether the input data (`d_i`) is valid.
